# User Registration Form with React and Formik

A simple React application demonstrating form handling using controlled components and Formik, with mock API integration for user registration.


 **Features**
- Form validation for username, email, and password.
- Integration with [JSONPlaceholder](https://jsonplaceholder.typicode.com/) mock API.
- Real-time validation feedback.
- Success and error alerts for form submission.


 **Technologies Used**
- React
- Formik (for form state management and validation)
- Vite (for project setup and development server)

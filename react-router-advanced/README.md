# Advanced Routing in React with React Router

A React application demonstrating advanced routing techniques using React Router. This project includes nested routes, dynamic routing, and protected routes for handling variable paths and authentication.

---

 **Features**
- Nested routes for managing sub-sections like Profile Details and Profile Settings.
- Dynamic routing for handling user-generated content, such as blog posts.
- Protected routes that require authentication before accessing certain pages.

---

 **Technologies Used**
- React
- React Router (v6)
- Vite (for project setup and development server)
